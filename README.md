# entry-task

### Running on a local system
#### 1. Run backend
```shell
cd .\backend\
npm install
npm run start
```

#### 2. Run frontend
```shell
cd .\frontend\
npm install
npm run dev
```

#### 3. Go to [http://localhost:8080/](http://localhost:8080/)
